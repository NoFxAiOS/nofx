# NoFx13 智能交易系统

基于官方 NoFx 核心的智能交易平台，集成用户认证和实时交易功能。

## 🚀 核心功能

### 官方交易引擎
- 实时市场数据监控
- 智能交易信号生成
- K线图表分析
- 多交易对支持

### 用户系统
- 安全注册/登录
- 用户会话管理
- 个性化仪表板
- 账户安全设置

### 技术特性
- 🐳 Docker 容器化部署
- 🔐 JWT 认证机制
- 📊 实时数据可视化
- 🌐 RESTful API 集成

## 🛠️ 技术栈

- **前端**: Streamlit
- **后端**: Python 3.11
- **数据库**: Supabase
- **部署**: Hugging Face Spaces
- **图表**: Plotly

## 📈 交易功能

- BTC/USDT, ETH/USDT 实时行情
- 智能买卖信号
- 风险控制指标
- 交易历史记录
