# NOFX 验收标准（初版）

## 接管阶段验收

### 文档
- [ ] 有中文项目总览
- [ ] 有中文架构说明
- [ ] 有模块索引
- [ ] 有开发日志
- [ ] 有待办与决策记录

### 基线
- [x] 后端测试可执行并通过
- [ ] 前端测试可执行
- [ ] 前端构建可执行
- [ ] 关键运行方式被记录

### 认知
- [ ] 已明确启动链
- [ ] 已明确决策链
- [ ] 已明确交易链
- [ ] 已明确风控链
- [ ] 已输出首轮风险清单

## 后续功能开发验收模板

每个功能至少回答：
1. 目标是什么
2. 改了哪些模块
3. 对哪些链路有影响
4. 如何测试
5. 是否更新文档/决策
6. 是否有回归风险
