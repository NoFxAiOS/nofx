# NOFX 风险清单（初版）

## 工程风险
1. 文档与实际代码可能有偏差
2. 前端当前未完成依赖安装与基线验证
3. 多交易所适配器带来较高的一致性维护成本

## 业务风险
1. 收益优化若脱离稳定性验证，容易得到虚假正反馈
2. 回测/仿真/实盘之间若逻辑不一致，结论不可靠

## 安全风险
1. API 密钥、钱包私钥、交易所凭证处理链必须专项审计
2. 默认配置（如 JWT secret 回退值）需检查生产风险
3. CORS 当前默认全开，后续需核验部署场景下的安全性

## 运行风险
1. 各交易所异常码、网络波动、状态同步延迟可能导致持仓不一致
2. 自动重试与补偿逻辑若处理不当，会造成重复下单或漏单
3. Telegram/Bot/外部服务故障可能影响辅助链路
