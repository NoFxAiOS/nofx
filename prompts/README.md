# Prompts

This directory contains prompts for use with AI models.

## meta_prompt.json

This prompt is designed to be used with Google AI Studio and NotebookLM.

### Usage

1.  Copy the contents of `meta_prompt.json`.
2.  Paste the prompt into one of the following websites:
    *   [Google AI Studio](https://aistudio.google.com/prompts/new_chat)
    *   [NotebookLM](https://notebooklm.google.com/)
3.  Upload the necessary materials and books to create the prompt.
