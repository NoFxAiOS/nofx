
```bash

```

应该看到:
```
[2025-11-02 xx:xx:xx] [INFO] 经济日历超精简版 - 启动中...
```

### 3. 查询数据库

```bash
python3 << 'EOF'
import sqlite3
db = sqlite3.connect('economic_calendar.db')
cursor = db.cursor()
cursor.execute("SELECT COUNT(*) FROM events WHERE importance = '高'")
print(f"✓ 高重要性事件: {cursor.fetchone()[0]}")
EOF
```
